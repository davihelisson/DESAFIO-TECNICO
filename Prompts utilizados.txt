DESAFIO 03
Criar um sistema para gerenciar agenda de contatos. Esse sistema deve ter:
1. Ter uma classe Contato com propriedades Nome, Telefone e Email.
2. Ter uma classe Agenda que armazene uma lista de contatos.
3. Permitir adicionar novos contatos, listar todos os contatos e buscar contatos por nome.
4. Permitir remover contatos da agenda.
